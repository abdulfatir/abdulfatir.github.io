Mobile Based Blood Analysis

Google Summer of Code, 2016

Abstract

The aim of this project is to use the camera and processing power of modern day cell phones to develop an intuitive and user-friendly application for the detection and concentration estimation of various biomarkers in blood sample images. It is later planned to be used as a screening test for cancer. The application will allow the user to take images of the blood samples in a set format. The image will then be segmented to detect the regions of interest. After noise removal, the intensity of each individual blob will be calculated. A linear curve will be fit through the intensity and known concentration data and the concentrations of the unknown samples will be estimated from the standard curve which will quantify the various molecules present in the sample.

Usage

Android Phone

  1. On your Android phone go to Settings > Security and Allow installation of Apps from Unknown Sources.
  2. Download and Install this .apk on your phone.
  3. Choose Load Image option and open image of the card with blood sample. Images of this dataset can be used. To be able to use the Take Picture option, you need to have a physical test card. You can get a printout of one of these images and use it instead for testing.
  4. Make sure that the image is loaded in portrait orientation. Rotate if necessary.
  5. Tap on Analyze. If the analysis is successful, the detected regions of interest will be highlighted using green color.
  6. Tap on each detected blob to enter its details. (Values to be entered for standard samples are shown in Fig.)
  7. Tap on Results and the estimated concentration of the unknown sample will be shown. The result will only be shown if the estimated concentration of the Quality Control Sample lies within 20% error bound.
